# Contributing

Thanks for your interest in contributing to Zinga.

## Project Status

Zinga is still under active development. You may encounter bugs, rough edges, or incomplete features.

If you find any issue or have suggestions, please report them. Feedback is very valuable at this stage.

## How to Contribute

- Open an issue for bugs, usability problems, or feature requests.
- Include clear reproduction steps, expected behavior, and actual behavior.
- If possible, attach logs, screenshots, or environment details.
- Keep pull requests focused and small when possible.

## Development

```bash
pnpm install
pnpm desktop:dev
```

Before opening a pull request, run:

```bash
pnpm lint
```
